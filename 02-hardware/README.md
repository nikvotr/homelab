# Hardware

In this section, I describe the hardware used for my homelab.

## Overview

The system is built using old but functional components.

The goal is to create a low-cost environment for learning and experimentation.

## Specifications

- CPU: Intel i3-4430 (4 cores, 4 threads)
- RAM: 6GB DDR3
- Motherboard: Gigabyte GA-B85M-D2V
- PSU: 400W

## Storage

- 240GB SSD -> operating system
- 2TB HDD -> main data storage
- 160GB HDD -> reserved for future use

## Networking

- Ethernet connection (LAN)
- Local IP address (static configuration)

## GPU

- Integrated Intel HD Graphics
- AMD Radeon HD 7750 (available for future use)

## Why This Hardware?

This setup is not powerful, but it is enough for:

- learning Linux system administration
- running lightweight services
- experimenting with Docker
- practicing cybersecurity concepts

## Limitations

- limited RAM
- older CPU architecture
- not suitable for heavy workloads

## Key Takeaways

- you do not need expensive hardware to start a homelab
- old hardware is enough for learning
- optimization is important on low resources

## Notes

This hardware setup demonstrates that practical experience can be gained even with limited resources.
